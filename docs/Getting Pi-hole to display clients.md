[As per this discourse post](https://discourse.pi-hole.net/t/how-do-i-show-hostnames-instead-of-ip-addresses-in-the-dashboard/3530?page=2), there are two ways to accomplish this:
1. use *Pi-hole's* built in DHCP server
2. use your *router's* DHCP server and modify `/etc/hosts` on your Pi-hole machine
